Weather API

How to Use

To use the Weather API, make a GET request to https://weather.mehmetfd.dev/api with the latitude and longitude of the location you're interested in. For example, to get the current temperature in Berlin, Germany, you could make a request to https://weather.mehmetfd.dev/api?lat=52.5200&lon=13.4050.

The API returns the current temperature in degrees Celsius, along with other weather data. If an error occurs, the API returns a JSON error object with a HTTP 400 status code.

For more information on how to use the API, please see the GitHub repository.
